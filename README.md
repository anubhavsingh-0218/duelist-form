# Duelist Creatives — Freelancer Intake System

A premium, Apple-style glass UI form to onboard creative freelancers across India.
## About

This is the Phase 1 hiring system for Duelist Creatives.

It allows photographers, editors, and creators to apply and join the Duelist network.

The goal is to build a fast, scalable onboarding pipeline for creative talent across India.

## Features

- Clean Apple-style glass UI
- Fully responsive (mobile + desktop)
- Formspree integration (no backend needed)
- Smooth scroll UX
- Minimal & fast loading
- Ready for scaling into full backend system

## Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- Vanilla JavaScript
- Formspree (Form backend)
- GitHub Pages (Hosting)
